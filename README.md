## Requirement

- node v12.13.0
- npm 6.12.0

## Installation

- `git clone https://github.com/jaimenunezl/events.git`
- `cd events`
- `npm install`
- `npm run start`

## Deploy viewer

When deployed into master branch automatically build into Netlify site.
